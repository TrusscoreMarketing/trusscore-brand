# Color

Reference swatches for design tools. The authoritative values live in SKILL.md Section 2.

| Name | HEX | Role |
|---|---|---|
| TC Slate | `#3A4B5C` | Primary — headings, structure |
| TC Text | `#111A22` | Body copy and captions only |
| TC Yellow | `#FEB100` | Accent only — never text on white |
| TC Grey | `#DFDBDA` | Borders, card fills, neutrals |
| TC Blue | `#009BD6` | Accent |
| TC Green | `#6BA543` | Accent |
| White | `#FFFFFF` | Backgrounds, reversed logo |

## Files

- `trusscore-brand-colours.cclibs` — Adobe Creative Cloud library. In an Adobe app, add it via Libraries to get the swatches in Illustrator, Photoshop, and InDesign.

- `trusscore.ase` — Adobe swatch exchange (Illustrator, InDesign)
- `trusscore.aco` — Photoshop swatches

Follow the 60-30-10 rule (Section 2).
