# Color

Reference swatches for design tools. Values are authoritative in the brand skill (Section 2); this folder holds the importable files.

| Name | HEX | Role |
|---|---|---|
| TC Slate | `#3A4B5C` | Primary — headings, structure |
| TC Text | `#111A22` | Body copy and captions only |
| TC Yellow | `#FEB100` | Accent only — never text on white |
| TC Grey | `#DFDBDA` | Borders, card fills, neutrals |
| TC Blue | `#009BD6` | Accent |
| TC Green | `#6BA543` | Accent |
| White | `#FFFFFF` | Backgrounds, reversed logo |

Add importable palette files here: `trusscore.ase` (Adobe), `trusscore.aco` (Photoshop), and a `swatches.svg` reference sheet. Follow the 60-30-10 rule (Section 2).
