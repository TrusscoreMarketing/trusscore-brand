# Fonts

Typeface files and licensing pointers. **Read the licensing note before committing any font file.**

## By context (brand skill, Section 7)

| Context | Typeface | Notes |
|---|---|---|
| Print / Premium | Circular Std | **Commercial license — do not commit the font files** |
| Microsoft Office | Aptos | Ships with Microsoft 365 |
| Web / Digital | Mulish (Muli Variable) | Open source — SIL Open Font License |

## Licensing

- **Circular Std** is a commercial licensed font from Lineto. Do not add `.otf`, `.ttf`, `.woff`, or `.woff2` files to this repository. Instead, direct licensed users to the internal purchase/download location. Add that link here.
- **Mulish** is licensed under the SIL Open Font License and may be committed. Download: https://fonts.google.com/specimen/Mulish
- **Aptos** is distributed by Microsoft with Office; users get it there rather than from this repository.

## Fallback chain (Section 7)

1. Circular Variable
2. Muli Variable
3. Mulish
4. Aptos

Arial and Georgia are not approved fallbacks.
