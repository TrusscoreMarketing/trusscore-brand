# Fonts

Typeface files and licensing pointers. **Read the licensing note before adding any font file.**

## By context (SKILL.md Section 7)

| Context | Typeface | Notes |
|---|---|---|
| Print / Premium | Circular Std | Commercial license — do not commit the font files |
| Microsoft Office | Aptos | Ships with Microsoft 365 |
| Web / Digital | Mulish (Muli Variable) | Open source — SIL Open Font License — committed here |

## Files

- `Mulish/` — Mulish variable fonts and static weights, plus the OFL license. Safe to commit and redistribute.

## Licensing

- **Circular Std** is a commercial font from Lineto. Do NOT add its files to this repo. Add the internal download link here for licensed users.
- **Mulish** is SIL OFL — included in `Mulish/`. Source: https://fonts.google.com/specimen/Mulish
- **Aptos** is distributed by Microsoft with Office.

## Fallback chain (Section 7)

1. Circular Variable
2. Muli Variable
3. Mulish
4. Aptos

Arial and Georgia are not approved fallbacks.
