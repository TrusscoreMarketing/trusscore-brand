# Logos (human library)

The full logo export set for the team — design, print, and web. These are the masters; the skill copies live in `skills/trusscore-brand-guidelines/assets/logos/` and should match.

Include every variant and format:

- **Variants:** primary, reversed (white), one-color white, one-color slate, wordmark-only (each variant), solo truss mark
- **Formats:** SVG (vector master), PNG (transparent, 1x/2x for web), EPS or PDF (print)
- **Color:** truss mark is TC Yellow `#FEB100`; wordmark is TC Slate `#3A4B5C` or white — never other colors

See the brand skill, Section 8, for clear-space, sizing, and do-not rules.
