# Logos (human library)

The full logo master set for the team. The skill bundles only a lean subset (see `skills/trusscore-brand-guidelines/assets/logos/`); everything lives here.

## Structure

- **`*.svg`** — vector masters (primary brand)
- **`png/1x`, `png/2x`, `png/3x`** — raster exports, including the favicon PNGs
- **`eps/`** — EPS vector versions for print
- **`programs/`** — sub-brand and program logos (see SKILL.md Section 8)

## Primary-brand masters

| File | What it is |
|---|---|
| `trusscore-logo-slate.svg` | Slate wordmark + yellow truss (primary, light backgrounds) |
| `trusscore-logo-white.svg` | White wordmark + yellow truss (reversed, dark backgrounds) |
| `trusscore-logo-onecolour-white.svg` | All-white lockup, for TC Yellow backgrounds |
| `trusscore-logo-tagline-slate.svg` | Slate lockup with "material + science" tagline |
| `trusscore-logo-tagline-reverse.svg` | White lockup with tagline |
| `trusscore-logo-tagline-onecolour-white.svg` | All-white lockup with tagline |
| `trusscore-truss-element.svg` | Solo truss mark (graphic accent) |
| `trusscore-truss-outline-element.svg` | Truss mark, outline |
| `t-icon-slate.svg` | T-icon, slate |
| `t-icon-white.svg` | T-icon, white |
| `t-icon-white-yellow.svg` | T-icon, white with yellow truss |

## Rules

- The truss mark may be used alone as an accent. The **T-icon may stand alone only as a favicon** — everywhere else it needs the full logo or "trusscore.com" nearby (SKILL.md Section 8).
- Truss mark is TC Yellow `#FEB100`; wordmark is TC Slate `#3A4B5C` or white — never other colours.
- Use these files as-is. Never recreate or restyle a logo.
