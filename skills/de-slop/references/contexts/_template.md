# Context profile template

A profile is thin. The company core already does the heavy lifting. A profile only holds what is specific to this kind of output. Copy this shape for each new profile. The best ones are built by distilling real examples, not by writing rules in the abstract.

## When this fires
One line. What output this profile is for, and the cue that picks it (modality, audience, where it lives).

## Register
How the voice shifts for this output, on top of the company core. Two or three lines, no more.

## Format and channel
What this output is expected to look like here. Length, structure, the channel norms. Only what is specific.

## What "done" looks like
For outputs with a job to do (a support reply, a proposal): the things it must cover to count as complete.

## On-profile example
A real, good example of this output. This is what Aligned looks like. (Distill from real ones.)

## Off-profile example
A real, weak version of the same thing, with what is wrong. This is what a flag looks like. (Distill from real ones.)
