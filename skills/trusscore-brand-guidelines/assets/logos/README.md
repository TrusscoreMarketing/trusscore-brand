# Skill logo assets

The lean subset of logo files bundled with the skill and referenced by SKILL.md. These ship inside the installed skill, so only what the skill needs lives here — the full library is in `brand/logos/`.

Bundled files:

- `trusscore-logo-slate.svg` — primary: slate wordmark + yellow truss (light backgrounds)
- `trusscore-logo-white.svg` — reversed: white wordmark + yellow truss; used for the dashboard header in Section 14c
- `trusscore-logo-onecolour-white.svg` — all-white lockup, for TC Yellow backgrounds
- `trusscore-logo-tagline-slate.svg` — slate lockup with "material + science" tagline
- `trusscore-logo-tagline-reverse.svg` — white lockup with tagline (PowerPoint footer)
- `trusscore-truss-element.svg` — solo truss mark (PowerPoint cover, Section 14b)

Rules:

- Use these files as-is. Never recreate a logo by styling text — see SKILL.md Section 8.
- The T-icon and sub-brand logos are intentionally not bundled here; they live in `brand/logos/` and `brand/logos/programs/`.
- Keep these in sync with the masters in `brand/logos/`.
